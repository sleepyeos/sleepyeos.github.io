PassCrawler



A small crawler script which queries a very large collection of leaked email:password and username:password combinations for any particular credential you might be interested in finding. At the time of writing, this database is comprised of some 3 billion credential sets, and is estimated to double in size over the next 30 days. It includes famous breaches such as AntiPublic, Collections #1-#5, Yahoo, as well as a small number which have been acquired via Google dorking. More information on these breaches can be found on security researcher Troy Hunt's blog or at haveibeenpwned.com.
