PKG Shark



A graphical utility which takes advantage of the Android Debug Bridge (ADB) to remove software from Android Devices. Many stock android devices come pre-loaded with bloatware and even spyware, and provide no facility to remove these applications from the device without root. By utilizing ADB, PKG Shark is able to remove these software packages for any user on the device, obviating the need for /system write privileges.
